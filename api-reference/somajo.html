somajo

SoMaJo specific functionality.

This module is based on SoMaJo. Use pip to install the necessary dependencies for this module: pip install mltb2[somajo]

extract_url_set(text: str) → Set[str][source]

Extract URLs from text.

An example:

from mltb2.somajo import TokenExtractor

token_extractor = TokenExtractor("de_CMC")
url_set = token_extractor.extract_url_set("Das ist ein Link: http://github.com")
print(url_set)

Example output:

{'http://github.com'}
Parameters:

text (str) – the text

Returns:

Set of extracted links.

Return type:
