InstallationÔÉÅ

MLTB2 is available at the Python Package Index (PyPI). It can be installed with pip:

pip install mltb2

For many modules optional dependencies are necessary. Those optional dependencies must be installed when you want to use the module. To install those module specific dependencies see the description of the individual modules in the API Reference.
